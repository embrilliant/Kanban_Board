## Kanban Board

#### Features
- Drag and drop image files from desktop to change the background of the board
- Drag and drop tickets
- Animated bin

#### Technology
  * JavaScript
     * Objects
     * jQuery
  * HTML5
     * Local storage
     * Drag and drop
  * CSS

#### Testing Framework
* Mocha
