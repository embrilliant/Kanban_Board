# Kanban Board

#### A beautiful minimal agile development tool for anything.

## Features
- Drag and drop image file from desktop to change the background of the board
- Drag and drop tickets
- Animated bin

#### Technology
  * JavaScript
     * Objects
     * jQuery
  * HTML5
     * Local storage
     * Drag and drop
  * CSS
     * Media query

#### Task Runner
* Grunt

#### Testing Framework
* Mocha

## Setup

Install external libraries:

```
bower install
```
